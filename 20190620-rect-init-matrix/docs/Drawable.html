getAABB(resultnullable) → (non-null) {Rectangle}

Get the rough axis-aligned bounding box for the Drawable. Calculated by transforming the skin's bounds. Note that this is less precise than the box returned by getBounds, which is tightly snapped to account for a Drawable's transparent regions. getAABB returns a much less accurate bounding box, but will be much faster to calculate so may be desired for quick checks/optimizations.
